Atalhos do Sistema

Quando disponíveis, as ações abaixo podem ser acessadas via teclas de atalho:

  • ESC: Cancelar/Fechar *
  • Ctrl + Enter: Salvar/Consulta cotação *
  • Ctrl + A: Adicionar
  • Ctrl + B: Buscar/Filtrar
  • Ctrl + D: Duplicar *
  • Ctrl + E: Editar *

* Necessita de um item selecionado.
